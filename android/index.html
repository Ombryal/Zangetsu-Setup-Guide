Step‑by‑Step Installation

Step 1: Download the APK

Scroll down to the Downloads section above and download the latest Android APK. If your browser asks whether you want to keep the file, tap Download anyway or Keep.

Step 2: Open the APK

Once the download finishes:

  • Tap the download notification, or
  • Open your Downloads folder using your file manager and locate the APK.

Tap the APK to begin installation.

Step 3: Allow app installation (if prompted)

The first time you install an APK from your browser or file manager, Android may block the installation.

If you see a message saying the installation isn’t allowed:

  1. Tap Settings.
  2. Enable Allow from this source (or Install unknown apps, depending on your Android version).
  3. Return to the installer.

This only needs to be done once for each app you use to install APKs (such as Chrome, Firefox, or your file manager).

Step 4: Install Zangetsu

Tap Install and wait for Android to finish installing the app. This usually takes less than a minute.

Step 5: Open Zangetsu

Once installation is complete:

  • Tap Open, or
  • Find Zangetsu in your app drawer or home screen.

You’re now ready to start using the app.

Updating Zangetsu

When a new version is released:

  1. Download the latest APK from this website.
  2. Open the new APK.
  3. Tap Update (your data and settings will be kept).

You don’t need to uninstall the previous version first.

Troubleshooting

“App not installed”

  • Make sure you’ve downloaded the correct APK for Android.
  • Ensure there’s enough free storage on your device.
  • If updating, make sure the APK is signed with the same certificate as your existing installation.

Installation is blocked

Enable Allow from this source for the browser or file manager you’re using.

Can’t find the downloaded APK

Check your Downloads folder or your browser’s download history.

Google Play Protect warning

Since Zangetsu is installed from outside the Play Store, Android may show a Play Protect warning. If you downloaded the APK from the official Zangetsu website or GitHub release, you can proceed with the installation.

System requirements

  • Android 5.0 (Lollipop) or later
  • At least 100 MB free storage
  • An internet connection for initial setup
