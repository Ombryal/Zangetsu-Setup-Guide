Steps

  1. Click the download button above – it will fetch the latest .exe installer or .zip portable build.
  2. If you downloaded an installer, run it and follow the setup wizard.
  3. If you got a portable zip, extract it to a folder of your choice and run Zangetsu.exe.
  4. Windows Defender may show a warning – that’s expected for unsigned apps; click “More info” then “Run anyway”.
  5. Launch Zangetsu and complete any initial configuration.
